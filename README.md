## ASMP
Aplikasi Sistem Management Perkantoran

## Fungsi
Aplikasi ini dibuat untuk mempermudah mengelola administrasi kantor, berbasis web. 

## Fitur
* Surat Masuk dan Keluar
* Notifikasi Surat Masuk
* Print Dokumen Surat Masuk/Keluar (PDF File)
* Log Aktivitas yang bisa di download
* Update App
* Paging
* PDF Layout Editor
* Manajemen Bidang/Bagian
* Manajemen Pengguna
* Dashboard
* Profile
* Profile Photo Gallery

## Penting
Aplikasi ini masih dalam tahap pengembangan, jadi aplikasi ini masih ada bug.
aplikasi ini sudah tidak dikembangkan lagi sejak (07/11/2019) dan rencana nya pengembang akan membuat aplikasi baru yang mirip, jika ada waktu.

## Persiapan
* Install XAMPP (https://www.apachefriends.org/download.html)
* Siapkan browser direkomendasikan memakai Chrome, jangan memakai browser Windows EDGE atau Windows Explorer
* Clone project ASMP dari (https://github.com/thisfikri/asmp)

## Penginstalan dan Konfigurasi Aplikasi
* Langkah 1: pindahkan clone project yang sudah di extract ke folder htdocs xampp, contoh: c:\xampp\htdocs
* Langkah 2: nyalakan server apache dan mysql, lalu buka phpMyAdmin di browser untuk membuat database aplikasi ASMP
* Langkah 3: jika database sudah dibuat, buka folder aplikasi asmp, lalu buka file database.php di nama_project/application/config/database.php, dan edit hostname, username, password, dan nama database sesuai dengan konfigurasi xampp kamu.
* Langkah 4: buka file nama_project/.htaccess dan ubah nama folder "asmp" pada "asmp/index.php" sesuai dengan nama_project
* Langkah 5: buka http://localhost/nama_project, nanti akan muncul halaman preregister dan kamu harus mengisi form di halaman ini, jika form sudah di submit dan berhasil maka penginstallan telah selesai dan aplikasi siap digunakan.
